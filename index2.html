¿Que es ServiWeb?

ServiWeb es uina red donde TODOS pueden ofrecer servicios. Una red donde todos los que buscan algun servicio peden encontratlo, contratarlo y calificarlo, para asi contribuir a la comunidad. ¿Como Contratar?

Si estas queriendo encontrar algun especialista que ofresca un servivio solo vas a tener que seguir unos simples pasos:

  • 1. Iniciar sesión (resgistrarse si es la primera vez que ingresa).
  • 2. Buscar el servicio que desea contratar.
  • 3. Elegir el usuario que mas le guste.
  • 4. CONTRATAR!

¿Como Publicar?

Si usted desea publicar un servicio en ServiWeb, para que la guente lo pueda contratar usted debe:

  • 1. Iniciar sesión (resgistrarse si es la primera vez que ingresa).
  • 2. Presionar el boton de PUBLICAR.
  • 3. Completar el formulario con los datos del servicio que desea ofrecer.
  • 4. Listo! Su servicio ya puede ser contratado.
